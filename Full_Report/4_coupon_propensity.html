Coupon redemption propensity

Guiding Question: Who to target (Coupon redemption propensity)

Quick answer: Target the top 30% of households ranked by our coupon redemption score. Start with TypeA campaignsthey convert about 2x more than TypeB/C. Prioritize recent, high-spend shoppers who already use coupons; they deliver the highest campaign spend and capture ~68% of redeemers. For dual goals (redemption + spend), fund the top score deciles that also show campaign spend uplift.

  • Why: The top three deciles capture ~68% of all redeemers and deliver about 2x lift versus random targeting.
  • How: Rank households per campaign by the model score, fund the highest-score TypeA campaigns first, then fill remaining budget with top-score TypeB/C households.
  • Who to target: Households with high spend and trip frequency in the last 60 days, recent purchases, and some coupon usage.

# Observational uplift: predicted spend gain from redemption (T-learner)
from sklearn.compose import ColumnTransformer
from sklearn.preprocessing import OneHotEncoder, StandardScaler
from sklearn.ensemble import GradientBoostingRegressor

uplift_preprocessor = ColumnTransformer(
    transformers=[
        ("num", StandardScaler(), numeric_cols),
        ("cat", OneHotEncoder(handle_unknown="ignore"), categorical_cols),
    ],
    remainder="drop",
)

uplift_preprocessor.fit(X)
X_all = uplift_preprocessor.transform(X)

is_redeemed = y == 1
is_not_redeemed = y == 0

reg_redeem = GradientBoostingRegressor(random_state=42)
reg_no_redeem = GradientBoostingRegressor(random_state=42)

reg_redeem.fit(X_all[is_redeemed], campaign_df.loc[is_redeemed, "sales_during_campaign"])
reg_no_redeem.fit(X_all[is_not_redeemed], campaign_df.loc[is_not_redeemed, "sales_during_campaign"])

campaign_df["pred_spend_if_redeem"] = reg_redeem.predict(X_all)
campaign_df["pred_spend_if_not"] = reg_no_redeem.predict(X_all)
campaign_df["pred_uplift"] = campaign_df["pred_spend_if_redeem"] - campaign_df["pred_spend_if_not"]

campaign_df["pre_spend_quartile"] = pd.qcut(
    campaign_df["sales_last_60d"], q=4, labels=["Low", "Mid-low", "Mid-high", "High"]
)

uplift_summary = (
    campaign_df.groupby(["pre_spend_quartile", "campaign_type"])
    .agg(mean_pred_uplift=("pred_uplift", "mean"), households=("household_key", "size"))
    .reset_index()
)

uplift_pivot = uplift_summary.pivot(
    index="pre_spend_quartile", columns="campaign_type", values="mean_pred_uplift"
)

fig, ax = plt.subplots(figsize=(8, 5))
sns.heatmap(uplift_pivot, annot=True, fmt=".0f", cmap="Blues", ax=ax)
ax.set_xlabel("Campaign type")
ax.set_ylabel("Pre-campaign spend quartile")
ax.set_title("Predicted spend uplift from redemption (T-learner)")
plt.tight_layout()

def friendly_label(feat: str) -> str:
    if feat.startswith("campaign_type_"):
        return f"Campaign type: {feat.split('_', 2)[-1]}"
    if feat.startswith("HOMEOWNER_DESC_"):
        val = feat.replace("HOMEOWNER_DESC_", "").replace("_", " ")
        return f"Homeowner status: {val}"
    if feat.startswith("KID_CATEGORY_DESC_"):
        val = feat.replace("KID_CATEGORY_DESC_", "").replace("_", " ")
        return f"Kids in HH: {val}"
    if feat.startswith("classification_"):

        grp = parts[1] if len(parts) > 1 else ""
        val = parts[2].replace("_", " ") if len(parts) > 2 else ""
        return f"Segment {grp}: {val}"
    if feat == "coupon_share_to_date":
        return "Historical coupon share"
    if feat == "tenure_days":

  • Deciles 1-3 are the sweet spot for balancing redemption and revenue; lower deciles add volume with less spend per household. Recommendation: Allocate coupons to deciles 1-3 first to capture both redemptions and revenue; only extend to deciles 4-5 if budget requires broader reach.

Data science notes & caveats:

  • Campaign-window spend and the uplift heatmap are observational (T-learner); they assume selection on observables and are not causal. Validate with randomized or quasi-experimental tests.
  • The model uses pre-campaign features only; campaign-window aggregates are excluded to avoid leakage, but unobserved factors may still bias results.
  • Results come from a single stratified split; use cross-validation and time-based splits to test stability before deployment.
  • Segment-level effects may shift with different coupon templates or seasonality; monitor and recalibrate as offers change.

Conclusion: Target the top three score deciles especially for TypeA campaigns to capture roughly two-thirds of all redeemers and the highest campaign spend. The model reliably ranks households above random (lift ~2x in top deciles), and high-spend, recent shoppers with coupon affinity respond best. Use the dual-objective chart to balance redemption volume and revenue when allocating limited coupon budgets.
